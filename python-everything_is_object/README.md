# Python - More Data Structures: Objects, References and Mutability

## Description

This project focuses on understanding how Python handles objects, references, assignments, aliases, mutability, and immutability.

The goal is not only to answer the tasks correctly, but also to deeply understand **why** Python behaves the way it does when working with variables and objects.

This project is different from typical coding projects because many tasks are based on reasoning about Python behavior rather than writing long scripts.

## Learning Objectives

By the end of this project, you should be able to explain:

- What an object is
- The difference between a class and an object (instance)
- The difference between mutable and immutable objects
- What a reference is
- What an assignment is
- What an alias is
- How to know if two variables are identical
- How to know if two variables point to the same object
- How to display the identifier of a variable
- What the built-in mutable types are
- What the built-in immutable types are
- How Python passes arguments to functions
